# wig-map

Interactive map of the WIG demonstrator test-site ranking — the **163 contender sites** (top ~150
worldwide + all of India). Each criterion score is anchored to a **cited external human/institutional
source** (QS, Heritage, WJP, World Port Index, AIS counts, OurAirports, named WIG craft) and
adversarially verified; the cited source is shown under every criterion in the popups. Search +
World-top-50 / India quick views. Charging is not required this phase.

Viewer only — the data pipeline, methodology, and the world-top-50 / India lists live in
https://github.com/crazyrealphysics/biz-artifacts/tree/main/wig-site-selection.

Live: https://crazyrealphysics.github.io/wig-map/
